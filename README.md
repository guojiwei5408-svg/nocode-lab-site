# 不会代码研究所 · 网站第一版

> 一个不会代码、不会剪辑、不会做自媒体的普通人，用 AI 做一些没啥用但很好玩的小东西，并记录学习、踩坑和成长。

## 一、项目介绍

「不会代码研究所」网站第一版，定位是 **研究所的大门 + 作品档案馆 + 第一个作品体验入口**。

它不是企业官网，不是教学站，也不是商业网站。核心原则只有一句：**先开门，再装修。**

- 简单大于复杂
- 完成大于计划
- 不登录、不数据库、不会员、不支付、不后台、不大型框架
- 纯 HTML + CSS + JavaScript，**第一版可以本地直接双击打开**

## 二、文件结构

```
不会代码研究所/
├── index.html         # 首页（研究所的大门）
├── works.html         # 作品列表（作品档案馆）
├── work-001.html      # 001 号作品《所长决定器》体验页
├── about.html         # 关于研究所 + 初心墙
├── inspiration.html   # 灵感停车场（本地脑洞记录本）
├── style.css          # 全站样式（所有 CSS 都在这里）
├── script.js          # 全站脚本（抽签 + 灵感停车场）
└── README.md          # 本说明文件
```

所有 CSS 集中在 `style.css`，所有 JavaScript 集中在 `script.js`，HTML 里几乎不写内联样式，方便用 Cursor / Codex 直接落地修改。

## 三、本地打开方式

最简单的方式：

1. 把整个文件夹保存到电脑任意位置。
2. **双击 `index.html`**，用浏览器（Chrome / Edge / Safari 均可）打开。
3. 顶部导航即可在「首页 / 作品 / 关于 / 灵感停车场」之间跳转。

> 提示：本项目不依赖任何服务器、不引入任何外部 CDN，断网也能用。
> 如果想用本地服务器预览（可选），可在文件夹内运行 `python3 -m http.server`，
> 然后浏览器访问 `http://localhost:8000`。

## 四、当前版本说明（v1 · 2026）

- **首页**：研究所大门，含标题、副标题、三大入口（🕹 玩点东西 / 📖 看看所长今天学了什么 / 💡 留个脑洞）、最新作品入口、核心理念。
- **作品列表**：001《所长决定器》已上线、002《文字人生模拟器》研究中、003 待定、004 AI 新玩意体验排队中。
- **001 所长决定器**：点击按钮从 **30+ 条研究任务** 中随机抽取，含研究日志、简化版 Prompt 思路、返回入口。
- **关于页**：所长是谁、为什么建研究所、不做什么、做什么、初心墙。
- **灵感停车场**：输入脑洞，使用浏览器 `localStorage` 本地保存、展示、删除单条、清空（带二次确认）。数据只在你本地，研究所看不到。
- **风格**：米白背景、暖橙强调、卡片 + 虚线边框、手账研究日志感，已做手机响应式适配。

## 五、暂不包含哪些功能（第一版有意省略）

- 登录 / 注册 / 会员
- 数据库 / 后端服务
- 支付 / 商业化入口
- 真正的投稿系统（灵感停车场目前只存本地浏览器）
- AI 实时生成（所长决定器目前是固定任务库）
- 排行榜 / 评论 / 复杂后台
- 任何大型前端框架（React/Vue 等）与外部 CDN

## 六、后续升级方向

1. **灵感停车场升级为真投稿**：接一个轻量后端或表单服务，让脑洞能真正提交给研究所。
2. **所长决定器接真 AI**：从固定任务库升级为调用 AI 实时生成研究任务 / 直接出小成品。
3. **作品逐个上线**：完成 002《文字人生模拟器》，并把 003、004 从「待定 / 排队」推进为真作品页。
4. **研究日志独立成栏目**：把「所长今天学了什么」做成可持续更新的博客式列表。
5. **细节体验**：抽签历史记录、分享卡片、暗色模式、更多表情与手账质感。
6. **逐步装修**：在「先开门」之后，按需要再补登录、收藏等能力——始终保持简单大于复杂。
