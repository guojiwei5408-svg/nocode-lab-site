# 不会代码实验室

一个科技感单页官网，用于「不会代码实验室」视频录屏素材。

## 当前定位

不会代码，也能用 AI 做点真东西。

主理人：老郭。  
目标：用 AI 边学边做，带刚接触或还没接触 AI 的人一起进步。

## 文件结构

```text
nocode-lab-site/
├── index.html
├── style.css
├── script.js
└── README.md
```

## 本地预览

方式一：直接双击 `index.html`。

方式二：在项目目录运行本地服务：

```bash
python3 -m http.server 8126
```

然后打开：

```text
http://127.0.0.1:8126/
```
